Graficadora de Funciones - Ayuda

Introducción

La Graficadora de Funciones es una herramienta que permite visualizar gráficamente funciones matemáticas. Utiliza la potente API de GeoGebra para realizar las representaciones gráficas de manera precisa e interactiva.

Cómo graficar una función

  1. Ingresa la función matemática en el campo de texto, utilizando x como variable independiente.
  2. Presiona el botón "Graficar" o la tecla Enter para generar la gráfica.
  3. La función aparecerá dibujada en el área de visualización.

Sintaxis básica

La graficadora reconoce la sintaxis matemática estándar:

Operación Símbolo Ejemplo
Suma + x + 3
Resta - x - 5
Multiplicación * 2 * x
División / x / 2
Potencia ^ x^2
Raíz cuadrada sqrt() sqrt(x)

Funciones disponibles

Función Sintaxis Descripción
Seno sin(x) Seno de x (en radianes)
Coseno cos(x) Coseno de x (en radianes)
Tangente tan(x) Tangente de x (en radianes)
Logaritmo natural ln(x) Logaritmo en base e
Logaritmo base 10 log(x) Logaritmo en base 10
Exponencial exp(x) o e^x Función exponencial e elevado a x
Valor absoluto abs(x) Valor absoluto de x

Ejemplos

Función cuadrática: x^2 + 2*x - 3

Función trigonométrica: 2*sin(x) + cos(3*x)

Función racional: (x^2 - 4)/(x - 2)

Función exponencial: e^x - 2 o exp(x) - 2

Nota: La variable independiente debe ser siempre x en minúscula.

Interacción con la gráfica

Una vez que la función está graficada, puedes interactuar con ella:

  • Zoom: Usa la rueda del ratón o los botones de zoom en la interfaz.
  • Desplazamiento: Haz clic y arrastra para mover la vista de la gráfica.
  • Inspección de puntos: Coloca el cursor sobre la gráfica para ver las coordenadas.

Funciones avanzadas

La graficadora también soporta funciones más avanzadas como:

  • Funciones por tramos
  • Funciones paramétricas
  • Ecuaciones implícitas

Consulta la documentación de GeoGebra para más detalles sobre estas funcionalidades avanzadas.
